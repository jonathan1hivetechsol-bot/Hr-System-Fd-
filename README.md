# Future Designz HR Panel

A comprehensive Human Resources and Payroll management system built with React, TypeScript, and Vite.

## Features

- **Employee Management**: Add, view, and manage employee information
- **Attendance Tracking**: Clock in/out system with attendance records
- **Leave Management**: Apply for and approve leave requests
- **Payroll Processing**: Generate and manage employee payroll
- **Dashboard**: Overview of HR metrics and pending tasks
- **Settings**: Configure company, payroll, and system settings

## Tech Stack

- React 19
- TypeScript
- Vite
- Bootstrap 5
- React Router DOM
- Local Storage for data persistence

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.
